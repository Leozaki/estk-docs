# eSTK.me 智能卡简介

eSTK.me 智能卡系列是由 eSTK.me Group 打造的一款的在任何仅支持 UICC 的设备使用 eSIM 的可插拔智能卡，致力于通过 STK 实现对 eSIM profile 进行简单管理，并改善 eSIM 在各平台下的可用性。

## 快速开始

**新的 eSTK.me 智能卡？是想怎么用呢。**

新到货？看看如何开始使用吧。

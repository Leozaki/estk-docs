# 激活 eSTK.me 卡片 STK 授权

下载并激活好 Profile 后，您需要打开手机内的 STK 套件，然后点开 "About"。您将看到类似以下的内容

```
内容占位符
```

::: warning
您如果在没激活配置文件就将卡片插入手机，您可能会看到 "无SIM卡" 提示。
:::
